GIT

Git — розподілена система керування версіями файлів та
спільної роботи. Проект створив Лінус Торвальдс для керування розробкою ядра Linux,
а сьогодні підтримується Джуніо Хамано (англ. Junio C. Hamano). Git є однією з найефективніших,
надійних і високопродуктивних систем керування версіями, що надає гнучкі засоби нелінійної розробки,
що базуються на відгалуженні і злитті гілок.

Запуск сайту в Інтернет

1. Створити нову гілку і переходимо на неї git checkout -b назва-гілки

2. З`єднати гілку в одну (merge)

3. Записати зміни в локальний репозиторій гілки dev

- Додати файли до iндексу git add назва файлу

- Записати зміни в локальний репозиторій гілки dev git commit -m"коментар"

- Записати локальний репозиторій гілки dev на сервер git push -u origin dev

4. Переходимо на гілку master куда ми хочемо додати код git checkout master

5. Обєднуємо гілку master із гілкою dev git merge dev

6. Записати локальний репозиторій гілки master на сервер git push -u origin master
